Reglas:

  • Cada año completo (12 meses) => $2000 por año, hasta un máximo de $10,000.
  • Los meses sobrantes (menos de 12) => $100 por mes.
  • El total por aguinaldo se suma al sueldo base para dar el total a pagar.
